void InsertionSort ( int  number[5])

Sorts an array of integers using the Insertion Sort algorithm.

This function sorts the given array of integers in ascending order using the Insertion Sort algorithm. It iterates through the array, considering one element at a time and inserting it into its correct position among the previously sorted elements.

Parameters
numberAn array of integers to be sorted.

The function modifies the input array in place.
